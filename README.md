## 📌 Project Overview

This project demonstrates an end-to-end Business Intelligence solution built using **SQL, MySQL, and Power BI**. The goal was to transform raw supply chain transaction data into meaningful business insights through SQL analysis and interactive dashboards.

---

## 🎯 Business Problem

Businesses generate thousands of transactions every day. Without proper analysis, it becomes difficult to identify sales trends, customer behavior, product performance, and market opportunities.

This project provides a data-driven solution by combining SQL analysis with interactive Power BI dashboards.

## 💡 Key Business Insights

- Europe generated the highest total sales.
- LATAM recorded the highest number of orders.
- Consumer customers generated the highest revenue.
- Fishing was the highest-performing product category.
- Field & Stream Sportsman 16 Gun Fire Safe was the highest-selling product.
- Africa recorded the lowest sales, highlighting opportunities for business expansion.
